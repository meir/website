I recently updated the webapp, I wanted an easier way to learn music notes with it. So in order to do this i added a Teacher Note. Its a note that you'll have to play, like a little game, when a teacher note moves, you have to play it, once you play it, the note moves again. This way you can learn the notes in a fun way.

The teacher can be enabled using the teacher button in the bottom left, above the theme button.

You can find the app on sweelinck.flamingo.dev.
