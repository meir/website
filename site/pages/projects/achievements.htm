Achievements

This project is a small program that will request the game youre playing on twitch and then check with steam what achievements you have for that game. I initially made it so i could learn Rust. I also wanted to make a program that would be useful for me since ive started streaming some games on twitch and trying to get 100% achievements on them. If youre interested in that, my twitch is here: {link:https://twitch.tv/YesImHuman__}.

If youre interested in the source code or want to use it for yourself, you can find it here: {link:https://github.com/meir/achievements}.
